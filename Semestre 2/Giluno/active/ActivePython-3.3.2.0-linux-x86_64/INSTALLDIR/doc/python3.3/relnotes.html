Release History and Changes (3.3)

Release date: 12-Sept-2013

New Features & Upgrades

  • First release of ActivePython 3.3, based on Python 3.3.2
  • PyPM 1.4.0
  • PyWin32 b218.3 (24 Mar, 2013)

Known Issues

This section lists known issues with ActivePython and Python.

Windows

  • The following Microsoft knowledge base articles may be helpful in solving MSI problems:
    • Q236597 - OFF2000: Error Message: The Installation Package Could Not Be Opened ...
      http://support.microsoft.com/kb/q236597/
    • Q224094 - OFF2000: Error Message: This Application Requires the Windows Installer to Run
      http://support.microsoft.com/kb/q224094/
    • Q247532 - Works 2000 Error Message: Installer Terminated Prematurely
      http://support.microsoft.com/kb/q247532/
    • Error 1303: The installer has insufficient privileges... (while running Office Setup)
      http://support.microsoft.com/kb/q228658/
    • Error 1316: Running Setup for Admin Installation
      http://support.microsoft.com/kb/q228592/
    • Error 1327: Invalid Drive During Office Installation
      http://support.microsoft.com/kb/q217666/
  • ActivePython 2.5 64-bit does not include PyWin32 and SSL support.

Mac OS X

  • ActivePython 2.5 does not include documentation (Bug #90508)

Linux

  • The hashlib module may not import on certain versions of RHEL and CentOS with SELinux enabled. See Bug #88464 for a workaround.
